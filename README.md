# House Price Prediction

## Project Overview
This project predicts house prices using Machine Learning techniques. The model is trained on housing data and uses various features to estimate property prices.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Steps Performed
1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Price Prediction

## Files
- house_price_prediction.ipynb : Main notebook
- README.md : Project documentation
